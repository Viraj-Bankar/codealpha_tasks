# Data Visualization Dashboard

This task focuses on transforming the analyzed laptop dataset into interactive visualizations to generate meaningful insights.

## Objectives
- Present laptop market insights visually
- Identify relationships between price, rating, and reviews
- Create clear and interactive dashboards

## Dashboard Insights
- Laptop price distribution
- Rating distribution across laptop models
- Relationship between price and rating
- Reviews vs rating analysis
- Brand-wise laptop comparison

## Tools Used
- Tableau
- Data Visualization Techniques
- Dashboard Design Principles
